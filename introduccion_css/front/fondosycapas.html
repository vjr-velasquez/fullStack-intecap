Propiedades de Fondos

Las propiedades de fondo en CSS permiten controlar la apariencia visual de un elemento.

  • background-color: Define el color de fondo del elemento.
  • background-image: Establece una imagen de fondo.
  • background-repeat: Controla si la imagen de fondo se repite o no.
  • background-attachment: Controla si la imagen de fondo se desplaza con el contenido o es fija.
  • background-position: Especifica la posición inicial de la imagen de fondo.

Propiedad: Opacity

La propiedad opacity controla la transparencia de un elemento, con valores de 0 (completamente transparente) a 1 (completamente opaco).
