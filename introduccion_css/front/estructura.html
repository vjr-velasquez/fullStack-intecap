  • Estructura de css
    • Selector: Es el elemento HTML al que se aplicarán los estilos. Puede ser una etiqueta, una clase, un ID, etc.
      Ejemplo: h1, .clase, #id.

    • Propiedad: Es la característica que deseas cambiar del selector, como el color, el tamaño, el margen, etc.
      Ejemplo: color, font-size, margin.

    • Valor: Es el valor que se le asigna a la propiedad.
      Ejemplo: red, 20px, 10%.

      • Ejemplo Completo 

      
                        h1 {
                            color: blue;
                            font-size: 24px;
                        }
  • Modelo de cajas en Css
    • El modelo de cajas es un concepto fundamental en CSS que describe cómo se estructuran y se dimensionan los elementos en una página web. Cada elemento HTML se considera una "caja" compuesta por cuatro partes

      1. Contenido (Content): El área donde se muestra el contenido (texto, imágenes, etc.).

      2. Relleno (Padding): Espacio entre el contenido y el borde.

      3. Borde (Border): Línea que rodea el contenido y el relleno.

      4. Margen (Margin): Espacio exterior que separa el elemento de otros elementos.

      Ejemplo con codigo

      
                        div {
                            width: 200px;
                            padding: 20px;
                            border: 5px solid black;
                            margin: 10px;
                        }
  • Elementos de linea y Elementos de bloque

    1. Elementos de linea

      • Ocupan todo el ancho disponible de su contenedor.

      • Comienzan en una nueva línea.

      • Ejemplos: div , p , h1 , a , h6 , ul , li

        • Ejemplo 

        
                                <div>Este es un elemento de bloque.</div>
                                <p>Este es otro elemento de bloque.</p>

    2. Elementos de bloque

      • Ocupan solo el espacio necesario para su contenido.
      • No comienzan en una nueva línea.
      • Ejemplos: span , a , strong , em , img

        • Ejemplo

        
                                <span>Este es un elemento de línea.</span>
                                <a href="#">Este es un enlace (también de línea).</a>
