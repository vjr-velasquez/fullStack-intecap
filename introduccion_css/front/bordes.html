Explicación sobre bordes en CSS


¿Qué es un borde en CSS?

Un borde es una línea que rodea un elemento HTML. Puedes personalizar su apariencia definiendo su estilo, color, grosor y, si lo deseas, redondear sus esquinas.

  1. Estilo de borde (border-style): Permite definir si el borde será sólido, punteado, doble, entre otros estilos.
  2. Color de borde (border-color): Determina el color del borde, ya sea usando nombres, códigos hexadecimales, RGB o HSL.
  3. Grosor de borde (border-width): Controla el grosor del borde mediante valores predefinidos o unidades específicas.
  4. Bordes redondeados (border-radius): Se utiliza para dar forma redondeada a las esquinas de los bordes.

Tipos de bordes en CSS


  1. none: No se muestra ningún borde.
  2. solid: Borde sólido.
  3. dotted: Borde de puntos.
  4. dashed: Borde de guiones.
  5. double: Borde dobl
  6. groove: Borde en relieve con efecto tallado.
  7. ridge: Borde en relieve opuesto a groove.
  8. inset: Borde empotrado (luce como si el elemento estuviera hundido).
  9. outset: Borde sobresaliente (luce como si el elemento estuviera elevado).

Colores de Borde


  • Nombre de color (red, blue, green, etc.).
  • Código hexadecimal (#ff0000).
  • RGB (rgb(255, 0, 0)).
  • HSL (hsl(0, 100%, 50%)).

    • Propiedad: border-color.


Bordes redondeados


  • A todos los bordes con un solo valor (border-radius: 10px).
  • Individualmente para cada esquina

    • border-top-left-radius
    • border-top-right-radius
    • border-bottom-left-radius
    • border-bottom-right-radius
