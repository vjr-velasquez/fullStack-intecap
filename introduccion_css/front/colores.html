Formas de especificar colores en Css


  1. Nombres de colores

    • CSS tiene una lista de 140 nombres de colores predefinidos (como red, blue, green, etc.).

      • Ejemplo


      
                        p {
                            color: red;
                        }
  2. RGB (RED , GREEN , BLUE)

    • Los colores se definen usando valores para rojo, verde y azul, en un rango de 0 a 255.

    • También se puede usar rgba, donde "a" es la transparencia (de 0 a 1).

      • Ejemplo


      
                        p {
                            color: rgb(255, 0, 0); /* Rojo */
                            background-color: rgba(0, 255, 0, 0.5); /* Verde con 50% de transparencia */
                        }
  3. HEX (Hexadecimal)

    • Los colores se definen usando un código hexadecimal de 6 dígitos, que representa los valores de rojo, verde y azul.

      • Ejemplo


      
                        p {
                            color: #ff0000; /* Rojo */
                            background-color: #00ff00; /* Verde */
                        }
  4. HSL (Hue , Saturation , Lightness )

    • Hue (tono): Un valor entre 0 y 360 (grados en el círculo cromático).
    • Saturation (saturación): Un porcentaje (0% es gris, 100% es color puro).
    • Lightness (luminosidad): Un porcentaje (0% es negro, 100% es blanco).

      • Ejemplo

      
                        p {
                            color: hsl(0, 100%, 50%); /* Rojo */
                            background-color: hsla(120, 100%, 50%, 0.5); /* Verde con 50% de transparencia */
                        }

    Te dejamos aca un ejemplo completo


    
                /* Usando nombres de colores */
                h1 {
                    color: blue;
                }

                /* Usando RGB */
                h2 {
                    color: rgb(0, 128, 0); /* Verde */
                }

                /* Usando HEX */
                h3 {
                    color: #ffa500; /* Naranja */
                }

                /* Usando HSL */
                h4 {
                    color: hsl(240, 100%, 50%); /* Azul */
                }

                /* Usando RGBA con transparencia */
                div {
                    background-color: rgba(255, 0, 0, 0.3); /* Rojo con 30% de transparencia */
                }

                /* Usando HSLA con transparencia */
                p {
                    background-color: hsla(300, 100%, 50%, 0.7); /* Morado con 70% de transparencia */
                }
